    Automated Class Attendance System with Image Processing and AI Integration

    Published: 2024 Co-authors: Hadi Fouani, Yehya Hatoum, Ray Bou Assy, Tina Yaacoub, Chantal Hajjar

    This paper presents an innovative automated attendance system leveraging facial recognition technology and machine learning algorithms. The system captures student images through classroom cameras, processes them using computer vision techniques, and automatically records attendance with over 95% accuracy. Our solution eliminates manual processes while addressing privacy concerns through secure data handling.

    Key Technologies:

    • OpenCV for image processing
    • DeepFace for facial recognition
    • Python Flask backend
    • React.js frontend dashboard
    • MongoDB for data storage
